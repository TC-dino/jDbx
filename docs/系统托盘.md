# 系统托盘中文支持说明

## 问题背景

在 Windows 平台上，使用 Java AWT 的 `SystemTray` 和 `TrayIcon` 显示系统托盘时，中文字符经常显示为方块（□□□）。这是因为：

### 根本原因

AWT 的 `PopupMenu`/`MenuItem` 在 Windows 上使用 Win32 GDI 原生渲染，字体回退机制对 CJK 字符不可靠。

```
FXTrayIcon (JavaFX MenuItem)
    ↓ AWTUtils.convertFromJavaFX()
AWT MenuItem → Windows 原生菜单渲染 → 使用系统默认字体 → 中文变方块
```

### 影响范围

| 组件 | 渲染方式 | 中文支持 |
|------|----------|----------|
| FXTrayIcon 上下文菜单 | 内部转为 `java.awt.MenuItem` → Win32 原生菜单 | ❌ 方块 |
| SystemTray 上下文菜单 | `java.awt.PopupMenu` → Win32 原生菜单 | ❌ 方块 |
| TrayIcon tooltip | `TrayIcon.setToolTip()` → Win32 原生提示 | ❌ 方块 |
| TrayIcon 通知 | `TrayIcon.displayMessage()` → Windows 气球通知 | ❌ 方块 |
| JavaFX 界面文字 | JavaFX 自有字体渲染引擎 | ✅ 正常 |

## 解决方案

### 方案：JavaFX 弹出窗口替代 AWT 菜单

**核心思路**：拦截托盘图标的右键点击事件，弹出一个 JavaFX Stage/Popup 作为右键菜单。这样所有文字都由 JavaFX 渲染，完美支持中文。

**优势**：
- 完全绕过 AWT 原生菜单的字体渲染问题
- JavaFX 对 CJK 字符有完美的支持
- 可以自定义菜单样式（圆角、阴影、动画等）
- 保持系统托盘功能（图标、提示、通知）

### 实现方式

1. 使用 AWT `SystemTray` + `TrayIcon` 显示图标（这是必须的）
2. 不使用 `TrayIcon.setPopupMenu()`（避免 AWT 菜单）
3. 拦截 `MouseListener.mouseClicked()` 检测右键点击
4. 在右键点击位置弹出 JavaFX `Popup` 作为菜单
5. 菜单项使用 JavaFX `MenuItem`，支持中文

## TrayContextMenu 公共组件

项目提供了 `dino.jdbx.TrayContextMenu` 公共组件，封装了 JavaFX 弹出菜单逻辑。

### 基本用法

```java
import dino.jdbx.TrayContextMenu;

// 创建右键菜单
TrayContextMenu menu = TrayContextMenu.create(primaryStage)
    .item("显示窗口", () -> primaryStage.show())
    .item("最小化", this::minimizeToTray)
    .separator()
    .item("退出", this::exitApp);

// 在右键点击时显示菜单
menu.show(screenX, screenY);
```

### 功能特性

| 功能 | 方法 | 说明 |
|------|------|------|
| 基础菜单项 | `item(text, action)` | 添加可点击的菜单项 |
| 禁用菜单项 | `item(text, action, true)` | 添加禁用的菜单项 |
| 图标菜单项 | `item(icon, text, action)` | 添加带图标的菜单项 |
| 分隔线 | `separator()` | 添加分隔线 |
| 自定义字体 | `font(font)` | 设置菜单字体 |
| 自定义样式 | `style(Style.LIGHT/DARK)` | 设置菜单样式 |
| 自定义宽度 | `minWidth(width)` | 设置菜单最小宽度 |
| 显示菜单 | `show(x, y)` | 在指定位置显示 |
| 隐藏菜单 | `hide()` | 隐藏菜单 |
| 清空菜单 | `clear()` | 清空所有菜单项 |

### 样式示例

```java
// 浅色样式（默认）
TrayContextMenu menu = TrayContextMenu.create(primaryStage)
    .style(TrayContextMenu.Style.LIGHT)
    .item("菜单项", () -> {});

// 深色样式
TrayContextMenu menu = TrayContextMenu.create(primaryStage)
    .style(TrayContextMenu.Style.DARK)
    .item("菜单项", () -> {});
```

### 完整示例

```java
// 带图标的菜单
TrayContextMenu menu = TrayContextMenu.create(primaryStage)
    .item(exitIcon, "退出", this::exitApp)
    .separator()
    .item("禁用项", () -> {}, true)  // 禁用状态
    .separator()
    .item("自定义项", () -> {})
    .font(Font.font("Microsoft YaHei", 14))
    .style(TrayContextMenu.Style.DARK);
```

## 使用示例

### SystemTrayExample

```java
// 初始化系统托盘
private void initSystemTray() {
    systemTray = SystemTray.getSystemTray();
    trayIcon = new TrayIcon(createDefaultImage());
    trayIcon.setImageAutoSize(true);
    trayIcon.setToolTip("jDbx 系统托盘");

    // 创建 JavaFX 右键菜单
    contextMenu = TrayContextMenu.create(primaryStage)
        .item("显示窗口", () -> { /* ... */ })
        .item("最小化到托盘", this::minimizeToTray)
        .separator()
        .item("退出", this::exitApp);

    // 添加鼠标监听器
    trayIcon.addMouseListener(new MouseAdapter() {
        @Override
        public void mouseClicked(MouseEvent e) {
            if (e.getClickCount() == 2) {
                // 双击：恢复窗口
                Platform.runLater(() -> {
                    primaryStage.show();
                    primaryStage.toFront();
                });
            } else if (e.getButton() == MouseEvent.BUTTON3) {
                // 右键：显示 JavaFX 菜单
                Platform.runLater(() -> {
                    contextMenu.show(e.getXOnScreen(), e.getYOnScreen());
                });
            }
        }
    });

    systemTray.add(trayIcon);
}
```

### FXTrayIconExample

```java
// 初始化托盘图标
private void initTrayIcon() {
    BufferedImage trayImage = createDefaultImage();

    // 使用 FXTrayIcon 创建托盘图标（不使用其菜单功能）
    trayIcon = new FXTrayIcon(primaryStage, trayImage);

    // 创建 JavaFX 右键菜单
    contextMenu = TrayContextMenu.create(primaryStage);
    buildContextMenu();

    // 获取底层的 AWT TrayIcon 并添加鼠标监听器
    java.awt.TrayIcon awtTrayIcon = trayIcon.getRestricted().getTrayIcon();
    awtTrayIcon.addMouseListener(new MouseAdapter() {
        @Override
        public void mouseClicked(MouseEvent e) {
            if (e.getClickCount() == 2) {
                Platform.runLater(() -> {
                    primaryStage.show();
                    primaryStage.toFront();
                });
            } else if (e.getButton() == MouseEvent.BUTTON3) {
                Platform.runLater(() -> {
                    contextMenu.show(e.getXOnScreen(), e.getYOnScreen());
                });
            }
        }
    });

    trayIcon.show();
    trayIcon.setTooltip("jDbx FXTrayIcon 演示");
}

/**
 * 构建右键菜单
 */
private void buildContextMenu() {
    contextMenu.clear();
    contextMenu
        .item("显示窗口", () -> { /* ... */ })
        .item("最小化到托盘", this::minimizeToTray)
        .separator()
        .item("退出", this::exitApp);
}
```

## 注意事项

### 托盘图标提示（Tooltip）

`TrayIcon.setToolTip()` 和 `TrayIcon.displayMessage()` 仍然使用 AWT 原生渲染，在某些系统上可能仍显示方块。如果需要完美支持，可以考虑：

1. 使用英文提示文本
2. 或者自定义托盘图标显示方式（如在图标上叠加文字）

### 字体设置

`TrayContextMenu` 默认使用 `Microsoft YaHei`（微软雅黑）字体，这是 Windows 系统自带的中文字体，确保中文正常显示。可以通过 `font()` 方法自定义字体。

### 跨平台兼容性

此方案在 Windows 上效果最佳。在 macOS 和 Linux 上，AWT 原生菜单通常能正常显示中文，但使用 JavaFX 弹出菜单可以提供一致的用户体验。
