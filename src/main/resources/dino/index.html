Bootstrap 3 界面原型详情

📋 本原型展示的 Bootstrap 3 核心特性：

  • 响应式栅格系统 — col-xs/sm/md/lg 四层级
  • 导航栏 — navbar + 下拉菜单 + 徽章
  • 面板组件 (Panel) — 带标题和多种样式
  • 轮播 (Carousel) — 自动播放公告
  • 进度条 (Progress Bar) — 多种颜色和状态
  • 表格 (Table) — 条纹+悬停+响应式
  • 标签 & 徽章 — Label + Badge
  • 警告框 (Alert) — 可关闭的多种提示
  • Well 组件 — BS3 特有的凹陷容器
  • 缩略图 (Thumbnail) — 产品展示网格
💡 提示： 还包括 Jumbotron、面包屑导航、分页、表单元素、Glyphicon 图标字体、 模态框等。所有组件均为 Bootstrap 3 原生特性，无需额外插件。
